Projects

Mapcreator

Mapcreator is a map editing tool that allows journalists, book publishers, broadcasters, etc. to easily create beautiful, content-rich, highly customizable maps. Results can be exported as raster or vector images or as embeddable HTML code. We even made interactive maps for live TV broadcasts in Canada!

Under the hood the tool uses Vue.js framework and a customized version of Mapbox GL JS map rendering engine.

2GIS map rendering engine

Together with a small team of ~3 developers, we built a WebGL map rendering engine from the ground up. It is capable of displaying very dense cartographic data with a high level of detail and remarkable performance.

Particular focus of the engine is on rich and high-quality visuals, which include floor plans embedded into the map, building entrances, high-definition 3D models for selected buildings, etc.

The engine is used in production on the main 2GIS website. Developer SDK is also available.

2GIS floor plan rendering engine

The first WebGL project made by 2GIS. Includes the rendering engine and a JavaScript API for embedding it into websites and mobile applications.
