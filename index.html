Projects

2GIS map rendering engine

Together with a small team of ~3 developers we built a WebGL map rendering engine from the ground up. It is capable of displaying very dense cartographic data with high level of detail and remarkable performance. Particular focus of the engine is on rich and high-quality visuals, which include floor plans embedded into the map, building entrances, high-definition 3D models for selected buildings, etc.

The engine is used in production on the main 2GIS website. Developer SDK is currently in alpha stage of development.

2GIS floor plan rendering engine

The first WebGL project made by 2GIS. Includes the rendering engine and a JavaScript API for embedding it into websites and mobile applications.
